Deployment Action Workflows

Asset availability state machine — Dragon Seats Inventory Management

Available Ready for use
Reserved Allocated to customer
Deployed In the field
Down Needs service
Reserve
Available → Reserved
  • Customer
  • Start Date
  • Away / Home
Deploy
Reserved → Deployed
  • Pick-up Date
  • Transport Vendor
Return
Deployed Reserved Down → Available
  • Return Warehouse
  • Condition
  • Status
Service
Available → Down
  • Condition
  • Notes
Transfer
Deployed → Deployed (new customer)
  • Customer
  • Start Date
  • Away / Home
Refurbish
Down → Down (tracks refurb)
  • Manufacturer
  • Return Date

Hover over arrows to highlight workflow paths. Dashed lines indicate alternate Return paths.

Happy path: Available → Reserve → Reserved → Deploy → Deployed → Return → Available

Service path: Available → Service → Down → (Refurbish) → Return → Available
